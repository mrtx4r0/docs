# Git --pretty=format:  実用的プレースホルダ一覧
git log, git showなどの--pretty=format: オプションで使えるプレースホルダの一覧<br>
※補足<br>
 --pretty=<format> の<format>はログ1件につき出力する形式を指定可能。<br>
"format:" (ユーザが自由に指定できる形式)を使用することが多いが、他にoneline, short, fullなどがある
なお、「:」をつけるのはformatだけ

## 🔹 基本情報（最重要）

| プレースホルダ | 意味 | よくある用途 |
|----------------|------|--------------|
| `%H` | フルコミットハッシュ | 一意の識別 |
| `%h` | 短縮コミットハッシュ | 履歴一覧 |
| `%T` | ツリーハッシュ（full） | ツリー比較 |
| `%t` | ツリーハッシュ（short） | 同上 |
| `%P` | 親コミット（full） | マージ解析 |
| `%p` | 親コミット（short） | 同上 |

---

## 🔹 Author（変更を書いた人）

| プレースホルダ | 意味 |
|----------------|------|
| `%an` | Author 名 |
| `%ae` | Author メールアドレス |
| `%ad` | Author 日付（ロケール形式） |
| `%aD` | Author 日付（RFC2822） |
| `%ar` | Author 日付（相対 "2 days ago"） |
| `%at` | Author UNIX timestamp |
| `%ai` | Author ISO8601 日付 |

---

## 🔹 Committer（コミットを記録した人）

| プレースホルダ | 意味 |
|----------------|------|
| `%cn` | Committer 名 |
| `%ce` | Committer メール |
| `%cd` | Commit 日付（ロケール） |
| `%cD` | Commit 日付（RFC2822） |
| `%cr` | Commit 日付（相対） |
| `%ct` | Commit UNIX timestamp |
| `%ci` | Commit ISO8601 |

---

## 🔹 コミットメッセージ

| プレースホルダ | 意味 |
|----------------|------|
| `%s` | 件名（subject） |
| `%f` | 件名（ファイル名向け整形） |
| `%b` | 本文（body） |
| `%B` | 件名＋本文（全文） |

---

## 🔹 ref（ブランチ・タグ）

| プレースホルダ | 意味 |
|----------------|------|
| `%d` | デコレーション（branch, tag など） |
| `%D` | デコレーション簡易版 |
| `%S` | リファレンス名（基本参照名） |

---

## 🔹 色付け

| プレースホルダ | 意味 |
|----------------|------|
| `%C(<color>)` | 色指定（red, green, yellow, blue, cyan, bold など） |
| `%Creset` | 色リセット |

---

## 🔹 その他

| プレースホルダ | 意味 |
|----------------|------|
| `%n` | 改行 |
| `%%` | `%` を表示 |
| `%N` | Notes（備考） |
| `%m` | マージ記号（<、>、-、空白） |
| `%G?` | GPG 署名状態（G, B, U など） |
